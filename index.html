Projects

PCOS Classification

This project focuses on the development and evaluation of a PCOS (Polycystic Ovary Syndrome) classification system using mutual information for feature selection. It involves a comparative analysis of classification models, with a particular emphasis on integrating Random Forest with advanced techniques such as Mutual Information for feature optimization, SMOTE-Tomek for addressing class imbalance, and effective outlier handling to enhance predictive accuracy and model robustness. The goal is to achieve a reliable and efficient classification framework for PCOS diagnosis.

AIDS Prediction

Coding is Love

The AIDS Virus Infection Prediction project focuses on building a predictive model using the Kaggle dataset. After thorough data preprocessing, including cleaning, feature selection, and balancing with SMOTE, models like Naive Bayes and Random Forest were implemented. Random Forest achieved the highest accuracy of 77.1% after optimization, demonstrating its effectiveness for AIDS classification.
